# CodSoft-DataScience-Internship

Data Science Internship Projects completed under the CodSoft Internship Program.

## Overview

This repository contains all the tasks completed during my Data Science Internship at CodSoft. The projects focus on data analysis, data preprocessing, machine learning, model evaluation, and predictive analytics using Python and popular data science libraries.

## Tasks Completed

### Task 1: Titanic Survival Prediction

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Logistic Regression
* Random Forest Classifier
* Model Evaluation using Accuracy, Confusion Matrix, and Classification Report
* Achieved 81.01% Accuracy

### Task 2: Movie Rating Prediction

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Linear Regression
* Random Forest Regressor
* Gradient Boosting Regressor
* Model Evaluation using MAE, RMSE, and R² Score
* Prediction of Movie Ratings based on Genre, Director, Actors, Votes, Duration, and Release Year
