# Titanic Survival Prediction

## Objective
Build a machine learning model to predict whether a passenger survived the Titanic disaster.

## Dataset
Titanic Dataset containing passenger details such as:

- Age
- Gender
- Passenger Class
- Fare
- Embarked
- Family Information

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Building
6. Model Evaluation

## Models Used

- Logistic Regression
- Random Forest Classifier

## Results

| Model | Accuracy |
|---------|---------|
| Logistic Regression | 81.01% |
| Random Forest | 81.01% |

## Conclusion

The machine learning models successfully predicted passenger survival with 81.01% accuracy. Gender, Passenger Class, Fare, and Age were identified as key factors influencing survival.
